# Class Names

## classes()

`returns` __`array`__

Fetches the css classes for the node as an array:

```javascript
rect.classes()
```

## hasClass()

`returns` __`boolean`__

Test the presence of a given css class:

```javascript
rect.hasClass('purple-rain')
```

## addClass()

`returns` __`itself`__

Adds a given css class:

```javascript
rect.addClass('pink-flower')
```

## removeClass()

`returns` __`itself`__

Removes a given css class:

```javascript
rect.removeClass('pink-flower')
```

## toggleClass()

`returns` __`itself`__

Toggles a given css class:

```javascript
rect.toggleClass('pink-flower')
```

----

Description: Fetches the css classes for the node as an array, test the presence of a given css class or adds a given css class.
