# Circular reference
Every element instance within SVG.js has a reference to the actual `node`:

## node

`returns` __`node`__

```javascript
element.node
```

## native()

`returns` __`node`__

The same can be achieved with the `native()` method:
```javascript
element.native()
```

## instance

`returns` __`element`__

Similarly, the node carries a reference to the SVG.js `instance`:

```javascript
node.instance
```

----

Description: Every element instance within SVG.js has a reference to the actual node. Similarly, the node carries a reference to the SVG.js instance.
